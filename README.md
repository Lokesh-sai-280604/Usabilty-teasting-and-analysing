# 🧪 Usability Testing & Analysis

A responsive web application built using **HTML, CSS, and JavaScript** to simulate the usability testing process for digital products. The application enables users to create test plans, collect participant feedback, record observations, and generate a usability analysis report.

---

## 📌 Project Overview

This project demonstrates the complete usability testing workflow, helping designers and developers evaluate how users interact with a digital product. It provides a simple interface to document testing sessions, collect feedback, calculate usability scores, and identify areas for improvement.

---

## ✨ Features

- 📋 Create a usability test plan
- 🎯 Define product goals and test scenarios
- 👥 Record participant feedback
- ⭐ Collect usability ratings (1–5)
- 📝 Store observations and improvement suggestions
- 📊 Calculate the average usability score automatically
- 📑 Display a usability testing report
- 💻 Responsive and easy-to-use interface

## 📖 How It Works

1. Enter the product name.
2. Define the testing goal.
3. Create a usability testing scenario.
4. Save the test plan.
5. Add participant details.
6. Record usability ratings, observations, and suggestions.
7. View the automatically generated usability report and average usability score.
